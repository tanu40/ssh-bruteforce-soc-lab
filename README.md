# SSH Brute Force SOC Lab

A hands-on SOC investigation lab built using Python Flask.

Students analyze Linux authentication logs to identify:
- SSH brute-force attacks
- attacker IP addresses
- compromised accounts
- successful logins
- indicators of compromise (IOCs)

---

# Technologies Used

- Python 3
- Flask
- HTML/CSS
- Linux auth logs

---

# Learning Objectives

Students will learn:

- SSH brute-force detection
- Log analysis
- IOC identification
- Incident investigation
- Authentication analysis
- SOC workflow basics

---


# Installation

## Linux / Kali

Install Flask:

sudo apt install python3-flask -y

---

# Running the Lab

Start Flask server:

python3 app.py

Open browser:

http://127.0.0.1:5000

---

# Investigation Questions

Students must identify:

1. Main attacker IP
2. Compromised username
3. Attack type

# Educational Purpose

This lab is intended for:
- cybersecurity education
- SOC analyst training
- blue-team exercises
- log analysis practice

Only use in authorized lab environments.
